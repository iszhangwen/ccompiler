### 逆后序

逆后序（Reverse Postorder, RPO） 是一种基于深度优先搜索（DFS）的节点排序方式，常用于 编译器数据流分析、拓扑排序 以及 图算法优化。
它的核心思想是： 在DFS中，节点在所有后继节点访问完成后才被加入序列，最后将该序列反转，即得到逆后序顺序。
它尽量保证在处理某个节点前，其所有前驱节点都已被处理，从而减少迭代次数，加快收敛速度。

### 最近公共祖先(LCA)算法

最近公共祖先（Lowest Common Ancestor，简称LCA）是指在一棵树中，两个节点的最近公共祖先是这两个节点的所有公共祖先中离根节点最远的那个
朴素算法的基本思想是通过不断向上跳转节点，直到两个节点相遇。具体步骤如下：
比较两个节点的深度，让深度较大的节点向上跳转，直到两个节点深度相同。
然后同时向上跳转，直到两个节点相遇，相遇的节点即为LCA。
